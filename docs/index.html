Read this before you deploy

wptoolsVai is a powerful admin tool. If an attacker reaches it, they control your server. Always do the following:

  • Change the default login. It ships as admin / password — replace it with your own hashed credentials.
  • Restrict access by IP using the built-in allow-list (and remove any example IPs).
  • Delete the file from your server the moment you're done using it.
  • Never commit wp-config.php, backups, or exports to Git.

Change the password

Generate a hash and paste it in — your plaintext password never touches the file:

php -r "echo password_hash('YOUR_STRONG_PASSWORD', PASSWORD_DEFAULT), PHP_EOL;"
define('USERNAME', 'yourname');
define('PASSWORD_HASH', '$2y$10$...your-generated-hash...');

Install in 4 steps

PHP 7.4+ with MySQLi. Works on shared & cPanel hosting.

Download

Grab tools.php and style.css from the repo.

Upload

Place both in your WordPress root (next to wp-config.php) or a subfolder.

Secure

Change the password and add your IP before opening it.

Use & remove

Open /tools.php, do your work, then delete the file.

git clone https://github.com/smtir/wptoolsVai.git
cp wptoolsVai/tools.php wptoolsVai/style.css /path/to/your/wordpress/

Usage guide

Click a tool to see how it works.

Deleting files & folders
  1. Open Delete → Browse and select items.
  2. In Advanced Options pick Dry Run (preview), Soft Delete (to Trash), or Backup (copy first).
  3. Run it — large batches process in chunks with a progress bar.
Safe domain migration (Search-Replace)
  1. Enter old & new values (e.g. https://old.com → https://new.com).
  2. Run with Dry Run to preview the count.
  3. Export the database first as a backup.
  4. Uncheck Dry Run and apply — it's serialization-safe.
Locked out? Recover admin access
  • List Admins to see all administrators.
  • Reset Password on any admin, or Create Admin to make a fresh login.
Back up / export the database

Click Export Database, wait for the table-by-table progress, then Download .sql. Do this before any destructive DB action.

Detect hacked core files

Click Check Core Integrity. It fetches official checksums for your exact WordPress version and lists any modified or missing core files — a strong sign of compromise.

Scan & quarantine malware

Click Start Scan, review flagged files, then Quarantine (reversible) or delete them.
